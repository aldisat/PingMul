# PingMul

Ping multiple IPs using Python. Works for Windows and Unix.

## How to
1. Create a file containing the list of ip's you want to ping
```
$ cat iplistexample.txt 
google.com
wikipedia.com
gasdasf.com
111.111.111.111
192.168.18.1
192.168.18.257
```
- Running pingmul.py
