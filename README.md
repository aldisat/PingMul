# PingMul

Ping multiple IPs menggunakan Python. Bisa digunakan untuk Windows dan Unix.

## How to
1. Masukkan ip target pada file `ip_list.txt`
2. Jalankan `python3 pingmul.py`

## Example
- Ip target
```
$ cat ip_list.txt 
google.com
wikipedia.com
gasdasf.com
111.111.111.111
192.168.18.1
192.168.18.257
```
